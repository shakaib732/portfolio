
# Portfolio Website

A Simple Static React.js Website to showcase my skills.

## Demo

https://shakaib732.github.io/portfolio


## Run Locally

Clone the project

```bash
  git clone https://github.com/shakaib732/portfolio.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
